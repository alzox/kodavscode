# Koda Theme for VS Code

A port of koda.nvim for Visual Studio Code containing:

- Koda Dark
- Koda Light
- Koda Moss
- Koda Glade

## Install

1. Open the extension folder in VS Code.
2. Use `Extensions: Install from VSIX...` after packaging, or load the folder as an extension workspace during development.

## Notes

The port mirrors the Neovim palette as closely as possible while mapping it to VS Code's workbench, editor, terminal, and semantic token color systems.
